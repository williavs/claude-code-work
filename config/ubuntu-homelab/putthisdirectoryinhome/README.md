# Tmux Scripts Setup

## Quick Start

Add these commands to your `~/.bashrc` file:

```bash
# Tmux wizard command
alias wiz='~/tmux-scripts/tmux-wizard.sh'

# Add other useful aliases here
```

## Apply Changes

After editing your `.bashrc`, reload it:

```bash
source ~/.bashrc
```

## Usage

Now you can just type:
```bash
wiz
```

And the tmux wizard will launch!
